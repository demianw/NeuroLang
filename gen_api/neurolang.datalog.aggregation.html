neurolang.datalog.aggregation module¶

Support for aggregations according to [1]. For instance the encoding of P(x, count<y>):-Q(x, y) in intermediate representation Implication(P(x, AggregationApplication(count, y), Q(x, y)) produces the set P(x, z) where z is the number of tuples (x, ...) in the set Q.

1

T. J. Green, S. S. Huang, B. T. Loo, W. Zhou, Datalog and Recursive Query Processing. FNT in Databases. 5, 105–195 (2012).
